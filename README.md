# Açelya'nın Yeri

Öğrenci olan kızım için hobi seviyesinde oyun, grafik ve görselleştirmeler.

## Giriş

| Kullanıcı | Şifre |
|-----------|--------|
| `acelya` | `ilovemyfather` |
| `serkan` | `ilovemydaughter` |

**Serkan** hesabıyla giriş yapınca üstte **📋 Bilgi** bağlantısı görünür; Açelya’nın giriş saatleri ve gezindiği sayfaların özeti burada.

## Firebase

Skor tablosu ve aktivite kayıtları için `shared/firebase-config.js` kullanılır. Realtime Database kurallarına `activity` dalını ekleyin (`shared/firebase-config.example.js` içinde örnek kurallar var).
