🔄 Yazılım Geliştirme Yaşam Döngüsü (SDLC)

SDLC, yazılımın fikirden canlıya uzanan yolculuğudur. Her aşamaya tıklayarak o aşamada hangi metodolojilerin (Waterfall, Agile, DevOps) nasıl çalıştığını görün.

🧱 SOLID Prensipleri

Robert C. Martin'in (Uncle Bob) formüle ettiği 5 temel nesne yönelimli tasarım prensibi. Her biri ❌ kötü ve ✅ iyi kod örneğiyle açıklanmıştır.

🎨 Tasarım Desenleri (Design Patterns)

Gang of Four (GoF) tarafından kataloglanan 23 klasik tasarım deseni. Her biri tekrar eden tasarım problemlerine zarif çözümler sunar. Üç ana kategoriye ayrılır.

🖱️ Bir tasarım deseni kartına tıklayın.

🌿 Git Branching Stratejisi — Git Flow

Git Flow, Vincent Driessen'in önerdiği popüler bir branching modelidir. main (production), develop (entegrasyon), feature/, release/ ve hotfix/ dallarıyla yapılandırılmış bir iş akışı sunar.

🧪 Test Piramidi & CI/CD Pipeline

Test Piramidi: Çok sayıda hızlı unit test, daha az integration test, az sayıda yavaş E2E test. CI/CD: Code → Build → Test → Release → Deploy → Monitor otomasyon zinciri.
