🏗️ Nesne Yönelimli Programlama — 4 Temel Direk

OOP, kodu nesneler (durum + davranış) etrafında organize eder.

🔒
Encapsulation (Sarmalama)
class BankAccount {
private double balance;
public void deposit(double a){
if(a>0) balance+=a;
}
}
🧬
Inheritance (Kalıtım)
class Animal { void eat(){...} }
class Dog extends Animal {
void bark(){...}
}
🎭
Polymorphism (Çok Biçimlilik)
Animal a = new Dog();
a.makeSound(); // Dog's version
a = new Cat();
a.makeSound(); // Cat's version
🎨
Abstraction (Soyutlama)
abstract class Shape {
abstract double area();
}
class Circle extends Shape {
double area(){ return π*r*r; }
}
